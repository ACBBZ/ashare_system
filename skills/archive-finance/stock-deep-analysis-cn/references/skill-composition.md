# 组合关系说明

这个 skill 不替代底层 5 个 skills，而是作为总控层统一编排：

## 1. akshare-open
负责：
- 综合数据底座
- 基本面增强模块
- 事件驱动模块
- 新闻总结模块
- 估值模块
- 策略模块
- 龙虎榜 / 资金流 / 板块 / 宏观 / 市场总貌

## 2. akshare-stock-data
负责：
- 基础行情与财务接口兜底
- 原始表格抓取
- 当 akshare-open 某增强模块缺数据时补基础接口

## 3. stock-analysis-cn
负责：
- 单票技术面与财务面解释模板
- 风险点与观察位的表述框架

## 4. finance-news-cn
负责：
- 新闻检索、摘要、催化核验
- 当 AKShare 新闻不足时，用网页实时检索补充

## 5. stock-monitor-anomaly
负责：
- 量能异动、换手率、振幅、涨跌幅异常
- 短线活跃度判断

## 总控原则
- 同一结论尽量由 2 个以上模块交叉支持
- 若模块冲突，优先保留原始数据并写出分歧
- 若模块失败，保留 `缺失说明` 而不是删掉整段分析
