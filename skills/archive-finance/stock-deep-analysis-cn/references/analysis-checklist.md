# 个股全面分析检查清单

## 必查模块
- [ ] 基础信息（代码、名称、行业、上市时间、市值）
- [ ] 基本面（营收、利润、ROE、毛利率、负债率、现金流）
- [ ] 技术面（趋势、均线、波动、阶段强弱）
- [ ] 新闻面（最近 3-10 条重点新闻）
- [ ] 事件面（公告、研报、机构调研、龙虎榜、回购、股东变化）
- [ ] 资金面（主力、个股、板块、北向）
- [ ] 量能异动（涨跌幅、换手率、量比、振幅）
- [ ] 板块面（行业/概念板块强弱）
- [ ] 大盘面（指数与市场情绪）
- [ ] 宏观面（按需）
- [ ] 估值面（如可得）
- [ ] 风险提示

## 结论必须回答
- 这只票当前属于什么类型：价值型 / 趋势型 / 题材型 / 事件驱动型 / 高波动博弈型？
- 当前主驱动是什么？
- 最值得继续跟踪的 3 个变量是什么？

## 数据缺失时
- 不停止分析
- 必须写清：缺了什么、影响什么判断
